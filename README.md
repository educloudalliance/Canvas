# Canvas

## Tarkoitus
Canvas avulla nähdään tilannekuva organisaation tai organisaatioiden alla tapahtuviin kehitysprojekteihin, joiden kehitystoiminta perustuu Githubin prosesseihin ja logiikoihin.
